# Self-Improving Agent

> 매일 스스로 제품을 만들고 개선하는 자율 에이전트

## 프로젝트 개요

이 에이전트는 사람이 개입하지 않아도 매일 조금씩 **실제 판매 가능한 서비스/제품**을 만들어 나갑니다.

에이전트가 하는 일:
1. **시장 조사** — 사람들이 원하는 것, 경쟁 도구, 틈새 기회를 분석
2. **방향 결정** — 만들 제품/서비스를 스스로 결정
3. **점진적 개발** — 매일 기능을 추가하고 코드를 개선
4. **PR 자동 생성** — 변경 이유와 내용을 설명하는 PR을 자동으로 생성

## 현재 상태

🔍 **시장 조사 및 제품 방향 탐색 중**

에이전트가 어떤 서비스를 만들지 스스로 조사하고 있습니다.
결정되면 이 README가 자동으로 업데이트됩니다.

## 에이전트 작동 방식

매일 오전 10시, 에이전트는:

1. 이전 작업 리뷰 (BACKLOG.md, DECISIONS.md 확인)
2. 코드베이스 분석 (코드 품질, 누락된 기능, 개선점 탐지)
3. 오늘 할 작업 1개 선택 (제품 개발에 직접 기여하는 작업)
4. 코드 변경 (최대 3개 파일)
5. PR 생성 (한국어, 변경 이유 포함)

## 철학

- **제품 우선**: 인프라·자동화 개선보다 실제 사용자에게 가치 있는 기능 우선
- **작게, 자주**: 매일 작고 완성된 변경 하나씩
- **투명성**: 모든 결정은 DECISIONS.md에 기록
