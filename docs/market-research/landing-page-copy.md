## 3. 주요 기능 3가지

### 기능 1: 완전 자율 제품 개발 (Product Autopilot)
**제목**: 시장조사부터 코드까지, 알아서 합니다
**설명**: 에이전트가 경쟁 분석, 기능 기획, 코드 작성, PR 생성까지 전체 루프를 자율 실행합니다. "이거 만들어"라고 지시할 필요 없이, 에이전트가 스스로 제품에 필요한 다음 작업을 찾아 실행합니다.
**아이콘 제안**: 로켓 또는 순환 화살표

### 기능 2: 매일 1PR, 꾸준한 성장
**제목**: 하루에 하나씩, 매일 제품이 나아집니다
**설명**: 매일 오전 실행되어 작고 완성된 변경 하나를 PR로 제출합니다. 큰 리팩토링이 아니라 매일 검토 가능한 크기의 개선. 머지 여부는 항상 당신이 결정합니다.
**아이콘 제안**: 캘린더 + 체크마크

### 기능 3: 투명한 의사결정 로그
**제목**: 왜 이 작업을 했는지, 항상 설명합니다
**설명**: 에이전트의 모든 결정은 DECISIONS.md에 기록됩니다. 무엇을, 왜 선택했는지 읽을 수 있어 블랙박스가 아닙니다. 에이전트를 신뢰하되, 검증할 수 있습니다.
**아이콘 제안**: 문서 + 돋보기

## 7. FAQ 섹션 초안

**Q: 에이전트가 만든 PR을 꼭 머지해야 하나요?**
A: 아닙니다. 모든 PR은 검토 후 머지 여부를 직접 결정합니다. 에이전트는 제안할 뿐, 최종 결정은 항상 개발자의 몫입니다.

**Q: 내 코드가 안전한가요?**
A: 에이전트는 GitHub App으로 연결되며, 필요한 최소 권한만 요청합니다. 코드는 분석 후 즉시 삭제되며 저장하지 않습니다.

**Q: 어떤 언어/프레임워크를 지원하나요?**
A: 현재 Python, JavaScript/TypeScript, React 프로젝트를 우선 지원합니다. 지원 언어는 지속적으로 확대 중입니다.

**Q: Devin이나 Copilot과 뭐가 다른가요?**
A: Copilot은 코드 자동완성, Devin은 지시받은 작업 수행에 특화되어 있습니다. Self-Improving Agent는 "무엇을 만들지"부터 스스로 결정하는 유일한 완전 자율 제품 개발 에이전트입니다. 가격도 Devin의 1/17 수준입니다.
