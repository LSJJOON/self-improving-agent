## 2. 경쟁 제품 비교

### 2-1. Devin (Cognition AI)
- **유형**: 완전 자율 AI 소프트웨어 엔지니어
- **가격**: $500/월 (250 ACU 기준), 실질적으로 작업당 $2 이상
- **타깃**: 중소·대기업 개발팀, 테크 스타트업 CTO
- **강점**
  - 웹 브라우저, 터미널, 코드 에디터를 스스로 조작
  - GitHub 이슈 → PR 완전 자동화
  - 공식적으로 "최초의 완전 자율 AI 소프트웨어 엔지니어"로 알려짐
- **한계**
  - 가격이 매우 높아 1인 개발자·인디 해커에게 부담
  - 복잡한 도메인 지식이 필요한 작업에서 자주 실수
  - 작업 당 감독·검토 필요 → 완전 방치 운영 어려움
  - 미국 외 지역 결제·서비스 접근 제한 있음

### 2-2. GitHub Copilot
- **유형**: AI 코드 자동완성 + 채팅 어시스턴트
- **가격**: 개인 $10/월, 비즈니스 $19/월/사용자
- **타깃**: 모든 규모의 개발자 (가장 대중적인 AI 코딩 도구)
- **강점**
  - GitHub과 완전 통합, IDE 플러그인으로 즉시 사용
  - 단일 라인 자동완성부터 함수 단위 생성까지 지원
  - 월 정액 요금으로 예산 예측 가능
- **한계**
  - **자율성 없음**: 개발자가 매 순간 직접 지시해야 함
  - 단순 반복 작업에는 효율적이나 기획·설계 판단 불가
  - 제품을 "스스로 만드는" 기능 없음

### 2-3. SWE-agent (Princeton NLP)
- **유형**: GitHub 이슈 자동 해결 오픈소스 연구 에이전트
- **가격**: 무료 (오픈소스)
- **타깃**: AI 연구자, 벤치마크 실험 목적
- **강점**
  - 학술 벤치마크(SWE-bench)에서 높은 정확도
  - Claude/GPT-4 백엔드 교체 가능
  - 완전 오픈소스, 커스터마이징 자유도 높음
- **한계**
  - **상업 서비스 아님**: 설치·운영이 복잡
  - UI/UX 없음, 일반 사용자 접근 불가
  - 제품 방향성 결정·시장조사 등 상위 레이어 기능 없음

### 2-4. AutoCodeRover (NUS)
- **유형**: 자동 버그 수정 연구 에이전트
- **가격**: 무료 (오픈소스)
- **타깃**: 연구자, 대규모 레거시 코드베이스 유지보수
- **강점**
  - 스펙트럼 기반 결함 위치 파악(SBFL) 기술 활용
  - 코드 구조 이해도 높아 복잡한 버그 수정 가능
- **한계**
  - 연구 프로토타입 수준, 프로덕션 사용 어려움
  - 버그 수정에 특화 → 신규 기능 개발 미지원
  - 관리·운영 UI 없음

### 2-5. Cursor
- **유형**: AI 통합 코드 에디터 (VS Code 포크)
- **가격**: 무료 Hobby, Pro $20/월, Business $40/월/사용자
- **타깃**: 전문 개발자, 팀
- **강점**
  - 가장 자연스러운 AI 코딩 경험 (현재 가장 인기 있는 도구 중 하나)
  - Tab 자동완성, 코드베이스 인식 채팅, 에이전트 모드
  - 기존 VS Code 워크플로와 거의 동일
- **한계**
  - **자율성 없음**: 여전히 개발자가 주도
  - 편집기 내부 작업에 국한, 외부 서비스 연동 제한
  - 제품 기획·시장조사 등 상위 작업 불가

### 2-6. Sweep AI
- **유형**: GitHub Issues → PR 자동 생성 SaaS
- **가격**: 오픈소스 무료, Pro $480/월 (팀 5명)
- **타깃**: 소규모 스타트업, 오픈소스 메인테이너
- **강점**
  - GitHub Issues에 댓글 하나로 PR 생성
  - 오픈소스 레포 무료 지원
- **한계**
  - 이슈 해결 중심, 제품 자율 개발 불가
  - Pro 플랜 가격 높음 ($480/월)
  - 서비스 지속성 불확실 (스타트업)

### 2-7. Aider
- **유형**: 오픈소스 CLI AI 코딩 도우미
- **가격**: 무료 (LLM API 비용만 부담)
- **타깃**: 개인 개발자, 자동화를 원하는 파워유저
- **강점**
  - 완전 오픈소스, 모든 LLM 사용 가능
  - 터미널에서 자연어로 코드 변경 요청
  - 활발한 커뮤니티
- **한계**
  - CLI 인터페이스, UI 없음 → 비개발자 진입 장벽 높음
  - **자율성 없음**: 작업마다 사람이 지시해야 함
  - 제품 전략, 시장조사 기능 없음

## 4. 타깃 사용자 분석

### 최우선 타깃: 1인 개발자 / 인디 해커
- **규모**: 전 세계 수십만 명 (Indie Hackers, Product Hunt 커뮤니티)
- **페인 포인트**:
  - 아이디어는 있지만 개발 시간이 부족
  - 코드를 짜면서 동시에 마케팅, 고객 발굴을 해야 함
  - 반복적인 기능 구현에 지쳐 번아웃
- **지불 의향**: 월 $29-99 (사이드프로젝트 도구에 익숙)
- **핵심 욕구**: "내가 자도 내 제품이 스스로 자라는 경험"

### 2순위 타깃: 스타트업 CTO (팀 1-5명)
- 비용 대비 효과를 따지는 합리적 구매자
- Devin($500/월)보다 저렴하고 특화된 대안을 원함
- 지불 의향: 월 $99-299

### 3순위 타깃: 오픈소스 메인테이너
- 이슈 처리, 작은 기능 PR에 에이전트 활용 가능
- 무료 플랜으로 유입 → 유료 전환 유도

## 5. 차별화 포인트 3가지

### ① 제품 자율 개발 (Product Autopilot)
기존 도구들은 "코드를 짜는 것"을 돕지만, Self-Improving Agent는 **"무엇을 만들지"부터 결정**한다.
시장조사 → 방향 결정 → 코드 개발 → PR 생성까지 완전 루프를 자율 실행.

### ② 인디 해커 가격대
Devin($500/월)의 1/10 이하 가격에서 "매일 1개 기능 자동 추가" 경험 제공.
월 $29-49 구간 — 인디 해커가 커피값처럼 쓸 수 있는 가격.

### ③ 투명한 의사결정 로그
에이전트가 왜 이 작업을 선택했는지 DECISIONS.md에 매일 기록.
사용자는 에이전트를 통제하는 느낌을 잃지 않으면서도 자율 실행 이점을 얻음.
