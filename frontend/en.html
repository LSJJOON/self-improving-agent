Frequently Asked Questions

Do I have to merge every PR the agent creates? ▾

No. Every PR is a suggestion. You review and decide whether to merge. The agent proposes; you always have the final say.

Is my code safe? ▾

The agent connects via a GitHub App and only requests minimum necessary permissions. Your code is deleted immediately after the task is complete and is never stored.

What permissions does the GitHub App request? ▾

Only the minimum permissions needed to create PRs: Code read/write (for creating PR branches), Pull Request write, and Metadata read. It does not access GitHub Secrets, Actions workflows, billing info, or org member settings. You can revoke app permissions at any time from GitHub Settings.

Is my code used to train AI models? ▾

No. Under our Anthropic Claude API agreement, your code is not used for AI training. Code is deleted immediately after PR creation, and ShipCrew does not retain your source code.

What data do you store? ▾

We store: Account email, generated PR links/titles/dates, agent run logs (auto-deleted after 90 days).

We do NOT store: Source code files, raw GitHub tokens, environment variables or secrets, full commit history.

What languages/frameworks are supported? ▾

Currently prioritizing Python, JavaScript/TypeScript, and React projects. Language support is continuously expanding.

How is this different from Devin or Copilot? ▾

Copilot specializes in code autocomplete. Devin executes tasks you assign. ShipCrew is the only fully autonomous product development agent that decides what to build on its own. And it costs 1/17th the price of Devin.
