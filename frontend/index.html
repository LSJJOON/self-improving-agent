자주 묻는 질문

에이전트가 만든 PR을 꼭 머지해야 하나요? ▾

아닙니다. 모든 PR은 검토 후 머지 여부를 직접 결정합니다. 에이전트는 제안할 뿐, 최종 결정은 항상 개발자의 몫입니다.

내 코드가 안전한가요? ▾

에이전트는 GitHub App으로 연결되며, 필요한 최소 권한만 요청합니다. 코드는 작업 완료 후 즉시 삭제되며 저장하지 않습니다.

GitHub App은 어떤 권한을 요청하나요? ▾

PR 생성에 필요한 최소 권한만 요청합니다: 코드 읽기/쓰기(PR 브랜치 생성용), Pull Request 쓰기, 메타데이터 읽기. GitHub 시크릿(Secrets), Actions 워크플로우, 결제 정보, 조직 멤버 설정에는 접근하지 않습니다. 언제든지 GitHub Settings에서 앱 권한을 취소할 수 있습니다.

코드가 AI 학습 데이터로 사용되나요? ▾

아닙니다. Anthropic Claude API 사용 계약상 코드는 AI 학습 데이터로 사용되지 않습니다. 코드는 PR 생성 작업 완료 후 즉시 삭제되며, ShipCrew 서버에 원본 코드를 보관하지 않습니다.

어떤 데이터를 저장하나요? ▾

저장하는 것: 계정 이메일, 생성된 PR 링크·제목·날짜, 에이전트 실행 로그 (90일 후 자동 삭제).

저장하지 않는 것: 코드 원본 파일, GitHub 토큰 원문, 환경 변수 및 시크릿, 커밋 히스토리 전체.

어떤 언어/프레임워크를 지원하나요? ▾

현재 Python, JavaScript/TypeScript, React 프로젝트를 우선 지원합니다. 지원 언어는 지속적으로 확대 중입니다.

Devin이나 Copilot과 뭐가 다른가요? ▾

Copilot은 코드 자동완성, Devin은 지시받은 작업 수행에 특화되어 있습니다. ShipCrew는 "무엇을 만들지"부터 스스로 결정하는 유일한 완전 자율 제품 개발 에이전트입니다. 가격도 Devin의 1/17 수준입니다.

언제 출시되나요? ▾
